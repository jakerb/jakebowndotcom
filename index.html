Projects.

FindaSeder.com

As the Jewish holiday Passover occured during the 2020 COVID-19 lockdown I was approached to help build an online resource to connect individuals spending this time alone to virtual Sedars hosted by others celebrating Passover across the world.

The site was built using WordPress and implemented custom functionality for public users to submit as a host or express an interest in joining a virtual Seder.

Writemd.xyz

Write.md is an easy to use markdown editor tool that automatically creates a shareable document URL to make sharing and collaboration quick and easy.

This is an active side project I developed when I couldn't find a distraction free word processor when writing project tenders and technical docs. Upon posting on Product Hunt Write.md gained #4 Product of the Day and currently hosts over 15,000 documents with 5,000 MAU's.

Appuccino

Appuccino is mobile application development framework I have been developing over a number of years. It allows developers to build completely bespoke hybrid apps for iOS and Android using WordPress as a backend and Javascript.

More recently Appuccino offers realtime app generation, certificate signing and asset generation all in the cloud. Appuccino is currently licensed under MyAppuccino.

Toolzilla.app

Using WordPress on a daily basis for the past 6 years I have amassed a library of snippets, generators and documentation that I have written and used in projects.

I then decided that I should offer my work to other WordPress developers in the community and so created Toolzilla. This is a relatively new side-project and I am spending my free time porting my code snippets and tools to the site.
